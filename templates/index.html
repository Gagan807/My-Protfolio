Projects

Project 1

Birth-Weight Predictor
A web-application built with Python that predicts newborn birth weight using a trained machine learning model. I developed both the backend API (using Flask or similar) and the front-end interface so users can input features (like gestational age, maternal factors, etc.) and receive predictions in real time. The model was trained on real data, serialized with pickle, and deployed in a way that makes it reusable and scalable. Inputs are validated, and dependencies are well-managed using a requirements.txt.


This project demonstrates my skills in:
  • Data Science — data cleaning, feature engineering, training and evaluating predictive models.
  • API development — exposing model predictions through a web API / app.
  • Full-stack web development — combining front-end and back-end.
  • Deployment & reproducibility — packaging dependencies, serializing models, and creating user interfaces.
